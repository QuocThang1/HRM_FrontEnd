# Human Resource Management (HRM) System - Frontend

**Project Timeline:** 08/2025 - 11/2025

**Backend Repository:** https://github.com/QuocThang1/HRM_BackEnd

## Overview
This repository contains the frontend source code for the Human Resource Management (HRM) System. The application provides a responsive and user-friendly interface designed to streamline internal operations, including employee records, task assignments, and payroll management. It also features interfaces for the AI-powered CV screening module.

## Technologies Used
* React.js
* RESTful API Integration

## Key Features
* **Dashboard & UI:** Intuitive interfaces for managing employees, tracking tasks, and viewing payroll details.
* **AI Interaction Module:** Dedicated components for uploading applicant CVs and displaying the automated screening results processed by the AI backend.
* **State Management:** Efficient handling of data flow and synchronization with the backend services.
